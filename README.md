# Pizza Delivery API - UDS Challenge

1. Bodyparser
2. Authentication
3. CORS
4. Lucid ORM
5. Migrations and seeds

### Migrations

Run the following command to run startup migrations.

```shell
$ adonis migration:run
```

### Start Application

To start application:

```shell
$ adonis serve --dev
```
